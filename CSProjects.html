Data Science Projects

In our CS Projects class, me and my classmates wrote a Python program that can read an uploaded donation file our school receives from Amazon and then return the total monetary amount of donations made. To do this, we had to take the ASIN numbers (Product IDs) from the Word Doc provided by Amazon, use an API to find the current price on Amazon, multiply that number by the quantity bought from the Word Doc, then add it to our total. My class went the API route since Amazon's website has a large amount of security to prevent web scraping.
